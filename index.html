Yay Tetris!

Hi there, I'm Liz Krane! Just for fun, I built this simple Tetris game "from scratch" with vanilla JavaScript and the web browsers' built-in Canvas API. See my GitHub repo for code and more details, or just play! (Controls listed below)

Controls

  • Left/right arrow keys: move left/right
  • Up arrow or X: rotate clockwise
  • Z or Ctrl key: rotate counter-clockwise
  • Down arrow: soft drop (move down faster)
  • Space bar: hard drop (instantly drop to bottom)
